Hello and thank you for buying my component. This tutorial will teach you how to customize this great component.

 

Install the component in your html

First of all, the rotator is reading the content and settings from the 2 xml files. In the content xml file you must put your content, including pictures and other swf. When placing all these files to your server you should respect the paths you added in the xml. If you are including this component in a CMS with a complex files structure like Wordpress i recommend you to put the full path for all the resources, pictures, swfs or the 2 xml files: http://www.domain_name.com/images/picture.jpg or http://www.domain_name.com/settings.xml.

From the v2.4 version you are able to send the content and xml files as parameters from the html to flash (red marked text below). In this way you will be able to use the same flash swf file in the same page for more than once, but using different content and settings. If you don't want to use this feature you don't need to add the red marked text but you need to use the default xml files: settings.xml and content.xml

The code for the embeded flash in your html page:

<script type="text/javascript">
AC_FL_RunContent( 'codebase','http://download.macromedia.com/pub/shockwave/cabs/flash/swflash.cab#version=9,0,28,0','width','950','height','250','src','header_v2_4?settingsxml=settings_v2_simple.xml&contentxml=content_v2_simple.xml','quality','high','pluginspage','http://www.adobe.com/shockwave/download/download.cgi?P1_Prod_Version=ShockwaveFlash','movie','header_v2_4?settingsxml=settings_v2_simple.xml&contentxml=content_v2_simple.xml' ); //end AC code
</script>
<noscript>
<object classid="clsid:D27CDB6E-AE6D-11cf-96B8-444553540000" codebase="http://download.macromedia.com/pub/shockwave/cabs/flash/swflash.cab#version=9,0,28,0" width="950" height="250">
<param name="movie" value="header_v2_4.swf?settingsxml=settings_v2_simple.xml&contentxml=content_v2_simple.xml">
<param name="quality" value="high">
<embed src="header_v2_4.swf?settingsxml=settings_v2_simple.xml&contentxml=content_v2_simple.xml" quality="high" pluginspage="http://www.adobe.com/shockwave/download/download.cgi?P1_Prod_Version=ShockwaveFlash" type="application/x-shockwave-flash" width="950" height="250"></embed>
</object>
</noscript>

If you want to change the flash dimensions just put in the above code the desired width and height in all the places (italic text)

Alsodon't forget to copy the "AC_RunActiveContent.js" file to your server and add the following code in the <head> section of your html:

<script language="javascript">AC_FL_RunContent = 0;</script>
<script src="AC_RunActiveContent.js" language="javascript"></script>

You can see all these details in the preview folder, index.html file.

 

Creating a slide

In the content.xml file you have the slides/ads. Here you can add as many as you want or you can edit them. For each ad / slide you can add as many pictures or SWF files as you want. An ad/slide looks like:

<ad theTime="7" theTitle="Iphone 3Gs">

<background gradient1="0x000000" gradient2="0x000000"></background>

<theText textTransparency="10" glowSize="10" glowColor="0xffffff" glowStrength="3" textColor="0xff0000" moveType="Strong" easingType="Out" transitionDuration="2" justFade="no" appearFrom="right" finalXposition="15" verticalAlign="top" useBackground="yes" backgroundColor="0x000000" backgroundTransparency="70" backgroundPadding="5"><![CDATA[<font color="#333333" size="+4">The perfect solution for your website header</font><br/><font color="#ffffff">Very easy to integrate in all the platforms. <b>FULL CUSTOMIZABLE</b></font>]]></theText>

<pictures>
<picture Xposition="0" Yposition="0" timeDelay="0" appearFrom="top" justFade="yes" useEffect="no" effect="Photo" theLink="http://www.YourDomainName.here" glowSize="10" glowColor="0xffffff" glowStrength="3" moveType="Strong" easingType="Out" transitionDuration="1">images/bg2.jpg</picture>
<picture Xposition="550" Yposition="25" timeDelay="1" appearFrom="left" justFade="no" useEffect="yes" effect="Photo" theLink="http://www.YourDomainName.here" glowSize="10" glowColor="0xffffff" glowStrength="3" moveType="Strong" easingType="Out" transitionDuration="1">images/product2.png</picture>
</pictures>

<link theTarget="_blank" btnName="Discover more" showBtn="yes" useLink="no" useTextsBackground="yes" >http://www.YourDomainName.here </link>

<mp3 player="on" file="MP3s/Ambiental.mp3" iconColor="0x000000"></mp3>

</ad>

For this one i will take each parameter and explain what it represents and how can be changed:

Text area:

  • theTime - represents how many seconds this slide will run
  • theTitle - represents the ad title. It will be displayed on the top when you put the mouse over a button at the bottom. The look and feel can be changed in settings.xml
  • <background gradient1="0xD6E324" gradient2="0x7D8F14"></background> - represents the slide's background. If you want to use a gradient enter the start and finish colors. If you want to have only one color put the same code for both gradient1 and gradient2. If you want to use a picture as background you can use the first picture in the pictures area, just metch the dimensions of the pictures according with your movie's dimensions and set Xposition and Yposition to 0.
  • theText - your text is html so it supports all kind of formatting but it needs to be put inside the CDATA Tag: <![CDATA[ YOUR TEXT GOES HERE ]]>
  • glowSize="10" glowColor="0xffffff" glowStrength="3" - for both text and picture you can add a glow effect. Here you can change the color and the glow's dimension and strength. If you put a dark color you'll obtain a shandow effect.If you don't want a glow or a shadow for your text or picture just put 0 for glosSize and 0 for glowStrength
  • moveType="Strong" easingType="Out" transitionDuration="2" - both text and picture have an initial movement when they appear on stage.Here you can customize that. For the moveType you have the following: "Strong", "Bounce" and "Elastic". For easingType you have: "In", "Out" and "InOut" (case sensitive). The duration represents how many seconds the movement will play.
  • justFade="no" - if you don't want any movement for the text and you want only a fade in effect set : justFade="yes". If you want the text to move in from left or right (check below) set justFade="yes"
  • appearFrom="right" - choose from which side the text will appear if you decide not to use only a fade in effect. "right" / "left"
  • finalXposition="15" - set the point where the text will stop when the movement is finished or where it will be placed if you want just fade in.
  • verticalAlign="middle" - define the vertical position for the text ("top" / "middle" / "bottom")
  • textTransparency="10" - set the transparency level for the text, 0 = transparent, 100=opaque
  • useBackground="yes" - put "yes" if you want to use a background for the text and "no" if not
  • backgroundColor="0x000000" - set the color for the text's background
  • backgroundTransparency="70" - set the transparency level for the text's background, 0 = transparent, 100=opaque
  • backgroundPadding="5" - the space in pixels between the text and the background's margins
  • textColor="0xff0000" - from here you can set the text's color. This is the overall color but if you want a word or sentence in another color, the html formating (<font color="#123456">text</font>) is still working.

Pictures area - General settings applying for all the pictures / swf files on this slide:

  • glowSize="10" glowColor="0xffffff" glowStrength="3" - you can add a glow effect. Here you can change the color and the glow's dimension and strength. If you put a dark color you'll obtain a shandow effect.If you don't want a glow or a shadow just put 0 for glosSize and 0 for glowStrength
  • moveType="Strong" easingType="Out" transitionDuration="2" - pictures and SWFs have an initial movement when they appear on stage (if you don't want them just to fade in, check below). Here you can customize that. For the moveType you have the following: "Strong", "Bounce" and "Elastic". For easingType you have: "In", "Out" and "InOut" (case sensitive). The duration represents how many seconds the movement will play.

Pictures area - Settings for each picture / swf:

  • Xposition="100" Yposition="0" - the final position . If there is movement these are the points where it will stop, if there is only fade in the picture / swf will be placed here
  • timeDelay="1"- you can set for a picture or swf to appear after a time delay. This value represents the delay in seconds.
  • justFade="no" - if you don't want any movement and you want only a fade in effect set : justFade="yes". If you want the picture to move in from left or right (check below) set justFade="yes"
  • appearFrom="top"- choose from which side the picture / swf will appear if you decide not to use only a fade in effect. You have: "right" / "left" / "top" / "bottom"
  • useEffect="no" effect="Photo" - if you want the picture / swf to run a special effect when appearing set useEffect="yes" and put the name of the effect you want here: effect="name here". You can see these special effects below in the settings.xml area.
  • from the version v2.3 you can add a link to each picture / swf: theLink="your link here" . If you don't want to have a separated link for each add just leave blank: theLink="".

Link area:

  • <link theTarget="_blank" btnName="Discover more" showBtn="yes" useLink="no" useTextsBackground="yes" >http://www.YourDomainName.here</link>- each ad has a link to an URL address. You can set theTarget where the link will open: _blank(new window / tab) or _parent (same window). By setting the "showBtn" parameter to "yes" on the current ad will appear a "Read more" button, too. You can change its text using btnName parameter. If you don't want this button to appear just put showBtn="no".
  • If you want to disable the general link on a slide put useLink="no". If you want to keep the link put useLink="yes"
  • If you are using a background for the text you can select to use the background for this button (useTextsBackground="yes" ) or use the background only for the text area and leave the button outside it (useTextsBackground="no" )

Mp3 Player

  • now you can add a mp3 file to each slide.
  • <mp3 player="on" file="MP3s/Ambiental.mp3" iconColor="0x000000"></mp3>
  • if you want the mp3 player enabled put player="on" else put player="off"
  • customize the color for the sound icon which will appear on the banner when the player is on.(iconColor="0x000000")

Settings:

<stageWidth>950</stageWidth>
<stageHeight>250</stageHeight> Here you must put the new dimensions for the header. Also you have to change them in yout html.
<startSlideShow>yes</startSlideShow> If you want the slide show to start automatically put here "yes", if not put "no".
<useRoundCornersMask>yes</useRoundCornersMask> You need to make the header a part of your site so if you want round corners put here "yes". If you want square angles put "no".
<roundCornerMaskColor>0xffffff</roundCornerMaskColor>If you selected round corners, here you can change their color to complete the look and feel
<showBottomButtons>yes</showBottomButtons> Show or hide the bottom buttons with the slides numbers (yes / no)
<showPlayPauseControls>yes</showPlayPauseControls> Show or hide the Play / Pause button (yes / no)
<fadeColor>0xffffff</fadeColor> When a slide is changing a nice fade effect is playing. You can change the fade color here.
<showTopTitle>yes</showTopTitle>Show or hide the top part with the ads title when the mouse is over a slide bottom button. (yes / no) Below you can change the look and feel.
<topTitleBackground>0x000000</topTitleBackground>
<topTitleBgTransparency>70</topTitleBgTransparency>
<topTitleTextColor>0xffffff</topTitleTextColor>
<showTimerBar>yes</showTimerBar>If you don't want the timer bar to appear set this value to "no"
<smallButtonsColor>0x121212</smallButtonsColor>
<smallButtonsNumberColor>0xffffff</smallButtonsNumberColor> Customize the look for the ads and pause / play buttons

<smallButtonsType>1</smallButtonsType> - Now you have 2 different small buttons types. Options: "1" / "2" . The second version is not using numbers, only smaller circle bullets.
<smallButtonsXoffset>20</smallButtonsXoffset> - Distance between left margin and the buttons
<smallButtonsYoffset>35</smallButtonsYoffset> - Distance between bottom margin and the buttons

<transparentBackground>no</transparentBackground> - if you want you Rotator to have a transparent background put "yes" (don't forget to put wmode="transparent" in your HTML). If you want to use the colored background, set "no"
